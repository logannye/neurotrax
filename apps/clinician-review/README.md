# Clinician review app

The review surface for the **Clinician Evidence Card**.

## First slice

- current and prior source clips;
- capture-quality summary;
- a small measurement table;
- provisional change and uncertainty;
- context and compatibility warnings;
- an `accepted` or `rejected` decision with an optional annotation.

Only accepted observations may enter longitudinal history.

## Non-goals

- diagnosis;
- treatment recommendations;
- order entry;
- patient messaging;
- EHR integration.
