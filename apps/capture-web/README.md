# Capture web sidecar

The browser sidecar for **Guided Capture**. The MVP accompanies a future
telehealth appointment but does not provide live calling.

## First slice

- consent and visible recording state;
- MacBook camera and microphone permission;
- local preview and microphone meter;
- seated speech and finger-tapping instructions;
- local task-bound recording;
- pass, one retry, or `not measurable`;
- encounter observation preview and deletion.

## Non-goals

- live telehealth calling;
- continuous recording;
- diagnosis;
- longitudinal analysis;
- clinical recommendations.
